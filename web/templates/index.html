How It Works

This tool uses Anthropic's Claude AI to convert Chef cookbooks to Ansible roles. The conversion process involves:

  1. Parsing Chef cookbooks and recipes
  2. Converting Chef code to Ansible using Claude AI
  3. Generating properly formatted Ansible roles
  4. Validating the generated Ansible code

Features

  • Convert entire Git repositories containing Chef cookbooks
  • Convert individual Chef recipes
  • Convert Chef recipe files
  • Generate properly formatted Ansible roles
  • Handle Chef notifications, templates, and resources
